This extension will help it behave more like a true app. You can use rules to decide which links the SSB should handle itself and which should be sent to your default browser. You can also choose to have incoming links opened in the main app window instead of in a new tab as Chrome ordinarily does.

The options page allows you to set up your rules, as well as import and export rules for backup or migration.

This extension is only intended for use with Apple OSX apps created with MakeChromeSSB. It will not do anything if you just install it in Chrome.
